# 🎮 Capítulo 2 - Experimento com Controles WPF

**Status:** 🔄 Em Desenvolvimento

## 📝 Descrição

Projeto focado em experimentação e aprendizado prático com diferentes controles WPF, explorando layouts, propriedades e eventos.

## 🎯 Objetivos do Projeto

- [ ] Explorar diferentes tipos de controles WPF
- [ ] Entender layouts e posicionamento
- [ ] Trabalhar com propriedades de controles
- [ ] Implementar event handlers
- [ ] Criar interfaces interativas

## 🛠️ Tecnologias Utilizadas

- **.NET 10** - Framework
- **C# 14.0** - Linguagem
- **WPF** - Windows Presentation Foundation
- **XAML** - Interface declarativa
- **Visual Studio 2026** - IDE

## 📚 Conceitos Explorados

- Controles WPF (Button, TextBox, Label, etc.)
- Layouts (Grid, StackPanel, etc.)
- Event Handling
- Data Binding (se aplicável)
- Estilos e Templates XAML

## 🚀 Como Executar

1. Navegue até a pasta do projeto:
   ```powershell
   cd Cap02-ExperimentWithControls
   ```

2. Abra a solution no Visual Studio:
   ```powershell
   start ExperimentWithControls.sln
   ```

3. Pressione **F5** para executar
